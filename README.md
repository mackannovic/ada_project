# ada_project
Applied Data Architecture Project - Tokyo Metro Routefinder (NEO4J/Python)

System Used: NEO4J and Python
Outputs expected:
data_load.py: Reads data from github sources and local files to build NEO4J db for analysis and rout planner
statistics.py:  Extracts basic stats for the Tokyo Metro System as gleamed from the data provided
planner.py: This is an interface that will present you with a map and allow you to select a to and from station for the Tokyo metro. This will then generate the shortest route with noted line transfers using a Dyjkstra Algorithm

This was developed on Python Version 3.11 on Windows 11 Pro
